# 🌌 Cosmic Diary

Cosmic Diary is a beautifully designed **cosmic-themed personal journal web app** that lets you write and store your thoughts privately. The app works **completely offline** and saves entries locally on your device, ensuring full privacy and security.

## ✨ Features

* 🔐 Password protected access
* 🌌 Cosmic themed interface
* 🎨 Day & Night modes with multiple themes
* 🌠 Animated space backgrounds (stars, planets, nebulae)
* 🎵 Optional background music
* 💾 Offline storage using IndexedDB
* 📱 Installable as a web app (PWA)
* 💻 Works on desktop and mobile browsers

## 🧠 Privacy First

Cosmic Diary stores all entries **locally in your browser using IndexedDB**.
No data is sent to any server. Your thoughts remain **100% private**.

## 🚀 How to Use

1. Download or clone the repository.
2. Keep all files in the same folder.
3. Open **Cosmic Diary.html** in your browser.

Optional:
Install it as an app by choosing **Add to Home Screen** in your browser.
